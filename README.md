# 科普动画知识图谱 · 地理地质

> Science Animation Knowledge Graph · Geology & Earth Science

## 项目简介

这是一个为地理科普动画《读地球的人》构建的知识图谱系统，包含：

- **49个节点**：概念、人物、书籍、事件、集数、素材
- **50条关系边**：科学关联、历史影响、动画需求
- **交互式可视化**：D3.js 力导向图，可搜索、筛选、拖动
- **深度研究资料**：5个领域的深度调研文件

## 快速开始

### 在线访问
访问 GitHub Pages：https://[用户名].github.io/science-animation-graph/

### 本地运行
```bash
git clone https://github.com/[用户名]/science-animation-graph.git
cd science-animation-graph
# 用浏览器打开 index.html
```

## 知识图谱结构

### 节点类型
| 类型 | 颜色 | 示例 |
|------|------|------|
| 概念 | 青色 | 均变论、K-Pg边界、板块构造 |
| 人物 | 橙色 | 莱伊尔、诺尔、阿尔瓦雷斯父子 |
| 书籍 | 粉色 | 《地质学原理》、《地球简史》 |
| 事件 | 紫色 | 小猎犬号航行、恐龙大灭绝 |
| 集数 | 蓝色 | 第一章、第二章、第三章 |
| 素材 | 红色 | AI动画、资料画面、院士采访 |

## 深度研究文件

项目包含5个深度研究文件：

1. **深度研究-前寒武纪.md** - 地质年代深度资料
2. **深度研究-人物传记.md** - 莱伊尔/诺尔/阿尔瓦雷斯传记
3. **深度研究-动画参考.md** - 视觉化最佳实践
4. **深度研究-人类世.md** - 科学争议与前沿
5. **深度研究-书籍研究.md** - 《地质学原理》《地球简史》深度书评

## 技术栈

- **可视化**：D3.js v7.8.5（力导向图）
- **数据格式**：JSON（graph.json）
- **部署**：GitHub Pages
- **研究工具**：Obsidian 知识库集成

## 项目背景

本知识图谱为纪录片《读地球的人》（翟明国院士主演）而建，用于：
- 科学概念的梳理与关联
- 动画素材的需求规划
- 脚本创作的知识支撑
- 科普内容的深度研究
