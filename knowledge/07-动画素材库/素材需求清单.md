# 动画素材需求清单

## 总体说明

本清单为科普动画纪录片《地质学家的眼睛》制作提供详细的视觉素材需求。清单分为三个部分：
1. **核心概念对应的动画类型**：每个科学概念的视觉表现方式
2. **具体的素材描述**：供动画师参考的详细描述
3. **已有素材 vs 待制作素材**：资源现状和制作优先级

## 二、具体的素材描述（供动画师参考）

### 1. 序章素材
#### 1.1 地球时间压缩动画
- **描述**：将46亿年地球历史压缩为24小时时钟
- **关键帧**：
  - 00:00:00 地球形成
  - 04:00:00 生命出现（蓝藻）
  - 21:00:00 多细胞生物
  - 23:30:00 恐龙时代
  - 23:59:59 人类出现（最后1秒）
- **视觉风格**：深空背景，发光时间线，关键事件图标弹出
- **技术需求**：3D时钟模型，粒子系统，动态文本

#### 1.2 岩石薄片制作流程
- **步骤分解**：
  1. 野外采样（地质锤敲击特写）
  2. 切割（金刚石锯片切割岩石）
  3. 磨制（研磨机将岩石磨至0.03mm）
  4. 粘片（加拿大树胶粘接玻璃片）
  5. 抛光（最终薄片在光下透明）
- **视觉风格**：2D步骤图，实拍工具特写，剖面展示
- **细节要求**：工具细节准确，厚度比例正确

#### 1.3 偏光显微镜奇幻世界
- **矿物展示**：
  - 石英：彩虹干涉色旋转
  - 长石：双晶纹如条形码
  - 云母：鲜艳的粉红/绿色
  - 橄榄石：高突起，裂纹特征
- **视觉风格**：抽象艺术，色彩饱和，动态变化
- **技术需求**：着色器编程，偏振光模拟

### 2. 第一章素材
#### 2.1 19世纪地质考察场景
- **场景元素**：
  - 莱伊尔手持地质锤和罗盘
  - 羊皮纸地图和素描本
  - 马车、望远镜、标本箱
  - 欧洲风景（阿尔卑斯山、意大利海岸）
- **视觉风格**：水彩画风格，柔和色调，怀旧感
- **细节要求**：服装、工具历史准确

#### 2.2 灾变论 vs 均变论对比
- **左侧（灾变论）**：
  - 诺亚洪水淹没大陆
  - 火山爆发，地震撕裂大地
  - 天使/神祇形象（象征超自然）
- **右侧（均变论）**：
  - 河流缓慢侵蚀峡谷
  - 沉积物一层层堆积
  - 地质学家观察测量
- **分割画面**：中间时间线分隔，左侧戏剧化，右侧平实
- **转场效果**：灾变场景逐渐淡化，均变场景成为主导

#### 2.3 "将今论古"方法论图解
- **流程图结构**：
  ```
  观察现代过程 → 识别特征 → 古代岩石中寻找相同特征 → 推断古代环境
  ```
- **案例展示**：
  - 现代河流交错层理 → 古代砂岩中的交错层理
  - 现代火山灰层 → 古代凝灰岩层
  - 现代冰川擦痕 → 古代冰碛岩擦痕
- **视觉风格**：简洁线条，箭头流动，案例插画

### 3. 第二章素材
#### 3.1 希克苏鲁伯撞击事件
- **阶段分解**：
  1. **逼近**：10公里小行星进入大气层
  2. **撞击**：瞬间汽化岩石，形成200公里陨石坑
  3. **冲击波**：环形冲击波向外扩散
  4. **抛射**：熔融岩石抛向太空
  5. **海啸**：千米高海啸横扫墨西哥湾
- **视觉风格**：电影灾难片级别，物理模拟准确
- **技术需求**：流体动力学模拟，粒子系统，体积光

#### 3.2 撞击冬天链式反应
- **系统图元素**：
  - 中心：撞击事件
  - 第一环：直接效应（火、震、海啸）
  - 第二环：大气效应（尘埃、酸雨、降温）
  - 第三环：生物效应（植物死亡、食物链崩溃）
  - 第四环：长期效应（生态系统重组）
- **视觉风格**：同心圆扩散，颜色表示严重程度
- **动态效果**：多米诺骨牌式连锁反应

#### 3.3 人类世地质证据
- **证据类型可视化**：
  - **放射性核素**：核爆蘑菇云 → 全球钚-239分布图
  - **塑料颗粒**：塑料生产曲线 → 海洋微塑料分布
  - **混凝土层**：城市扩张时间推移 → 全球夜间灯光变化
  - **物种灭绝**：背景灭绝率 vs 当前灭绝率对比图
- **视觉风格**：数据驱动，图表动画，实拍叠加
- **情感基调**：冷静呈现，避免说教

### 4. 第三章素材
#### 4.1 透明地球与板块构造
- **地球剖面结构**：
  - 内核（固态铁镍）：金色
  - 外核（液态铁镍）：橙色流动
  - 下地幔：深红色缓慢对流
  - 上地幔/软流圈：红色对流环
  - 岩石圈板块：灰色刚性块体
- **板块运动展示**：
  - 离散边界：洋中脊岩浆上涌
  - 汇聚边界：板块俯冲，山脉隆起
  - 转换边界：板块侧向滑动
- **技术需求**：体积渲染，流体模拟，透明材质

#### 4.2 地幔对流发动机
- **对流环细节**：
  - 热柱上升：从核幔边界到地表
  - 冷板下沉：从海沟到深地幔
  - 物质循环：彩色示踪粒子显示路径
- **热成像风格**：蓝（冷）→ 红（热）渐变
- **时间压缩**：数亿年循环压缩为30秒
- **尺度提示**：旁边标尺显示深度（0-2900km）

#### 4.3 一亿年后地球预测
- **大陆运动动画**：
  - 当前大陆位置（2025年）
  - 1000万年后：微小变化
  - 5000万年后：明显移动
  - 1亿年后：全新格局
- **预测场景**：
  - **场景A**：大西洋继续扩张
  - **场景B**：非洲-欧洲碰撞，地中海闭合
  - **场景C**：新超大陆形成
- **不确定性表达**：概率云图，多可能性展示

## 三、已有素材 vs 待制作素材的分类

### 已有素材库（可直接使用或修改）

#### 实拍素材
1. **地质学家工作场景**
   - 显微镜操作（4K，多角度）
   - 野外勘察（无人机航拍+地面拍摄）
   - 实验室分析（延时摄影）
   - 院士访谈（绿幕抠像备用）

2. **自然景观**
   - 峡谷侵蚀过程（延时，1年拍摄）
   - 火山喷发资料画面（购买版权）
   - 冰川运动（卫星影像时间序列）
   - 河流沉积特写（高速摄影）

3. **历史资料**
   - 19世纪地质插画扫描（公共领域）
   - 莱伊尔手稿数字版（博物馆授权）
   - 早期地质仪器照片（档案库）

#### 3D模型库
1. **地球模型**
   - 高精度地球纹理（NASA数据）
   - 板块边界矢量数据
   - 地形高程模型

2. **矿物晶体模型**
   - 常见造岩矿物（石英、长石等）
   - 晶体结构动画（生长过程）
   - 偏光效果着色器

3. **科学仪器模型**
   - 偏光显微镜（可拆解展示）
   - 地质锤、罗盘等工具
   - 实验室设备（质谱仪等）

#### 2D图形资源
1. **信息图表模板**
   - 地质年代表矢量图
   - 科学流程图元件
   - 数据可视化模板

2. **风格化插画**
   - 19世纪科学插画风格
   - 简约线条图解风格
   - 水彩背景纹理

### 资源协调建议

1. **团队分工**：
   - 3D团队：地球模型、灾难模拟、流体动力学
   - 2D团队：人物动画、历史场景、图解
   - 数据团队：科学可视化、图表动画
   - 特效团队：微观特效、转场效果

2. **技术标准**：
   - 分辨率：4K主片，2K备用
   - 帧率：24fps（电影感）或30fps（电视感）
   - 色彩空间：Rec.709（电视），DCI-P3（影院）
   - 输出格式：ProRes 4444（制作），H.264（交付）

3. **质量控制**：
   - 科学准确性审核（每阶段）
   - 视觉风格统一检查
   - 叙事节奏测试
   - 观众理解度测试

4. **文件管理**：
   - 版本控制（Git LFS）
   - 资产命名规范
   - 渲染农场队列管理
   - 备份策略（每日增量，每周全备）
